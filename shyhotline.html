Shy Hotline

An automated mobile application for shy people asking questions in Slack

Introduction

During my software engineering boot camp, I was assigned an automation project. The requirements for this project are as stated:

  1. Get Input data
  2. Authenticate the inputted data
  3. After verifying information, store it in a spreadsheet
  4. Once added to the spreadsheet an event is triggered for a notification
  5. A notification is sent to the slack channel
  6. Have someone respond to slack
    • If someone responds close the ticket
    • If no response after a certain time, send another notification
  7. Students Should add an extra component that allows creativity

Regarding the brief list of requirements for this project, I went above and beyond and dipped my toe into uncharted waters. Throughout my tech journey, I always wanted to explore mobile app development. Specifically, hybrid app development. What better way to learn something new than to learn and apply the skill in 2 weeks? For my application, Shy Hotline, I utilized the most resourceful hybrid app development framework, React Native. React Native is a Facebook-powered open-source hybrid app development framework. The motto "Learn once, Write everywhere," saves them time and money when developing mobile apps. My project, initially titled "Shy Guys" and now changed to "Shy Hotline", is a mobile automation application for those persons who are too shy to ask questions directly. This allows for less tension for people who possess a level of social anxiety.

Challenges & The Process

Some of the challenges I had to overcome involved the learning curve of learning React Native as well as refreshing my memory on JavaScript at the same time. Additionally, I had a rather hard time coming up with an idea that implemented all the requirements needed to qualify my project. Through one-on-one help with my boot camp instructor, various tutorials, and trial and error I was able to accomplish this project in a timely manner with no real experience with React Native and little JavaScript experience outside of web development.
